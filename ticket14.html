1. Перечислите категории встроенных функции в SQL. Приведите примеры функций каждой категории. Укажите примеры практического использования этих функций

Категории встроенных функций:

  • Агрегатные: SUM, COUNT, AVG, MIN, MAX
  • Строковые: LENGTH, CONCAT, SUBSTRING, LOWER, UPPER
  • Математические: ABS, ROUND, CEIL, FLOOR
  • Дата и время: NOW, CURDATE, DATEDIFF, MONTH

Примеры использования:

  • SELECT AVG(price) FROM products — средняя цена товаров
  • SELECT CONCAT(first_name, ' ', last_name) FROM users — объединение имени и фамилии
  • SELECT DATEDIFF(CURDATE(), birth_date) FROM users — возраст в днях

2. Опишите принцип работы, преимущества и недостатки динамической маршрутизации. Приведите примеры протоколов динамической маршрутизации и охарактеризуйте их

Динамическая маршрутизация — автоматическое построение маршрутов с помощью специальных протоколов.

Преимущества:

  • Автоматическая адаптация к изменениям в сети
  • Упрощение администрирования в больших сетях

Недостатки:

  • Более высокая нагрузка на оборудование
  • Задержки при пересчете маршрутов

Примеры протоколов:

  • RIP — простой протокол, основанный на количестве переходов
  • OSPF — использует алгоритм Дейкстры, более эффективен в больших сетях
  • BGP — протокол междоменной маршрутизации, используется в Интернете

3. Дайте определение понятию «компьютерный вирус». Перечислите пути заражения компьютера вирусами. Опишите признаки заражения компьютера вирусами

Компьютерный вирус — это вредоносная программа, способная самостоятельно распространяться и наносить вред системе.

Пути заражения:

  • Через вложения в электронных письмах
  • Скачивание файлов из ненадежных источников
  • Съемные носители (USB)
  • Уязвимости в ПО

Признаки заражения:

  • Замедление работы компьютера
  • Самопроизвольный запуск программ
  • Появление неизвестных окон и сообщений
  • Изменение или исчезновение файлов
