Билет 15

1. Дайте определение понятию «вычислительная сеть». Перечислите и охарактеризуйте основные типы сетей

Вычислительная сеть — это система, состоящая из взаимосвязанных компьютеров и других устройств, обеспечивающих совместное использование ресурсов и обмен данными.

Основные типы сетей:

  • LAN (локальная): охватывает небольшую территорию, высокоскоростная.
  • MAN (городская): объединяет несколько локальных сетей в пределах города.
  • WAN (глобальная): охватывает большие расстояния, используется интернет-технологиями.
  • PAN (персональная): используется для связи устройств одного пользователя (например, Bluetooth-сети).

2. Дайте определение понятию «драйвер». Укажите и охарактеризуйте типы драйверов

Драйвер — это программное обеспечение, обеспечивающее взаимодействие операционной системы с аппаратными устройствами.

Типы драйверов:

  • Драйверы устройств — для работы принтеров, видеокарт, сетевых карт и т.п.
  • Фильтрующие драйверы — добавляют функциональность (например, антивирусные фильтры).
  • Виртуальные драйверы — эмулируют работу физических устройств (например, виртуальный CD-ROM).

3. Укажите назначение агрегатных функций. Перечислите и охарактеризуйте агрегатные функции в SQL

Агрегатные функции — это функции в SQL, выполняющие вычисления над группами строк и возвращающие одно итоговое значение.

Основные агрегатные функции:

  • COUNT() — подсчет количества строк
  • SUM() — сумма значений
  • AVG() — среднее значение
  • MIN() — минимальное значение
  • MAX() — максимальное значение

Пример: SELECT AVG(salary) FROM employees; — средняя зарплата сотрудников
