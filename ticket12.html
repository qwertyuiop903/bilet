Билет 12

1. Дайте определение понятиям «первичный ключ» и «внешний ключ». Укажите и охарактеризуйте виды связей между таблицами в реляционных БД

Первичный ключ — это уникальный идентификатор записи в таблице. Он не может содержать NULL и должен быть уникальным.

Внешний ключ — это поле, указывающее на первичный ключ другой таблицы, обеспечивая связь между таблицами.

Виды связей:

  • Один к одному (1:1) — каждой записи в первой таблице соответствует одна запись во второй
  • Один ко многим (1:N) — одной записи первой таблицы соответствует несколько записей второй
  • Многие ко многим (M:N) — реализуется через промежуточную таблицу

2. Дайте определение понятию «протокол». Охарактеризуйте протоколы «IP», «TCP», «UDP»

Протокол — это набор правил, по которым происходит обмен данными в сети.

  • IP (Internet Protocol) — обеспечивает адресацию и маршрутизацию пакетов по сети.
  • TCP (Transmission Control Protocol) — обеспечивает надёжную доставку данных, контроль ошибок, подтверждения получения.
  • UDP (User Datagram Protocol) — передаёт данные без установления соединения, быстро, но без гарантии доставки.

3. Что понимается под понятием «командная строка»? Перечислите основные команды для работы с каталогами и файлами и охарактеризуйте каждую

Командная строка — интерфейс, позволяющий управлять системой с помощью текстовых команд.

Основные команды:

  • cd — смена текущего каталога
  • dir (Windows) или ls (Linux) — просмотр содержимого каталога
  • mkdir — создание нового каталога
  • del (Windows) или rm (Linux) — удаление файлов
  • copy (Windows) или cp (Linux) — копирование файлов
  • move (Windows) или mv (Linux) — перемещение файлов
