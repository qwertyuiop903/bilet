Билет 19

1. Дайте определение понятию «видеокарта». Опишите разновидности видеокарт. Опишите основные характеристики видеокарты.

Видеокарта — аппаратное устройство, предназначенное для обработки и вывода графической информации на экран монитора.

Разновидности видеокарт:

  • Интегрированные видеокарты — встроены в материнскую плату или процессор, используют системную память.
  • Дискретные видеокарты — отдельные устройства с собственной видеопамятью и мощным графическим процессором (GPU).
  • Внешние видеокарты — подключаются отдельно, например, через Thunderbolt, для повышения графической производительности.

Основные характеристики видеокарты:

  • Графический процессор (GPU): производительность и архитектура.
  • Объем видеопамяти (VRAM): количество памяти для хранения текстур и буферов.
  • Тип памяти: GDDR5, GDDR6, HBM и др.
  • Разрядность шины памяти: влияет на пропускную способность.
  • Частота работы GPU и памяти: влияет на скорость обработки графики.
  • Поддерживаемые технологии: DirectX, OpenGL, аппаратное ускорение.

2. Дайте определение понятию «диспетчер задач». Опишите основные функции диспетчера задач.

Диспетчер задач — системная утилита ОС, позволяющая контролировать работу процессов, ресурсов и состояния системы.

Основные функции диспетчера задач:

  • Просмотр запущенных приложений и процессов.
  • Мониторинг использования процессора, памяти, диска и сети.
  • Завершение или приостановка процессов.
  • Управление автозагрузкой программ.
  • Просмотр информации о производительности системы.

3. Дайте определение понятия «индекс». Опишите принцип работы индексов. Перечислите и охарактеризуйте виды индексов.

Индекс — структура данных, используемая в базах данных для ускорения поиска и сортировки информации.

Принцип работы: индекс создаёт упорядоченный указатель на строки таблицы, позволяя быстро находить данные без полного сканирования таблицы.

Виды индексов:

  • Первичный индекс: основан на первичном ключе, уникален для каждой строки.
  • Вторичный (непервичный) индекс: создаётся по другим столбцам, может быть неуникальным.
  • Уникальный индекс: обеспечивает уникальность значений в столбце.
  • Составной индекс: индекс по нескольким столбцам.
  • Полнотекстовый индекс: для быстрого поиска текста в больших объёмах данных.
