Билет 1

1. Дайте определение понятиям «база данных», «система управления базами данных (СУБД)». Перечислите и охарактеризуйте основные функции СУБД.

База данных (БД) — это организованная совокупность данных, предназначенная для хранения, изменения и обработки информации

Система управления базами данных (СУБД) — программное обеспечение для работы с базами данных

Основные функции СУБД

  • Управление данными (добавление, удаление, изменение, поиск)
  • Управление транзакциями (целостность данных)
  • Обеспечение безопасности и разграничения прав доступа
  • Резервное копирование и восстановление данных
  • Поддержка многопользовательского доступа и параллелизма

2. Перечислите и охарактеризуйте задачи канального уровня. Приведите примеры протоколов канального уровня и дайте их подробную характеристику.

Канальный уровень отвечает за передачу данных между узлами сети по физическому каналу

Задачи канального уровня

  • Обнаружение и коррекция ошибок передачи
  • Упаковка данных в кадры
  • Управление доступом к среде передачи
  • Обнаружение границ кадров

Примеры протоколов

  • Ethernet — протокол для проводных сетей с методом CSMA/CD
  • PPP — протокол точка-точка для соединения по последовательному каналу
  • HDLC — протокол управления потоком и коррекции ошибок

3. Дайте определение понятию «система управления доступом». Перечислите и охарактеризуйте основные аспекты системы управления доступом в Windows.

Система управления доступом — это набор механизмов, определяющих, кто и какие действия может выполнять с ресурсами

Аспекты системы управления доступом в Windows

  • ACL — списки контроля доступа для объектов
  • Права пользователей и групп — настройка доступа к файлам, программам и функциям
  • Аудит безопасности — журналирование попыток доступа
  • Роли и политики — централизованное управление через GPO
  • UAC — защита от выполнения опасных действий
