Билет 7

1. Дайте определение понятию «реляционная БД». Охарактеризуйте структуру реляционной базы данных

Реляционная база данных — это БД, в которой данные организованы в виде таблиц (отношений), каждая из которых состоит из строк и столбцов.

Структура реляционной БД включает:

  • Таблицы — основные объекты хранения данных
  • Строки (записи) — представляют отдельные экземпляры данных
  • Столбцы (поля) — представляют атрибуты записей
  • Первичные и внешние ключи — для идентификации записей и установления связей между таблицами

2. Дайте определение понятиям «коллизия пакетов», «CSMA/CD», «CSMA/CA». Укажите отличие в процессе обработки коллизий между CSMA/CD и CSMA/CA

Коллизия пакетов — ситуация, когда два устройства передают данные по сети одновременно, вызывая их искажение.

CSMA/CD (Carrier Sense Multiple Access with Collision Detection) — метод, при котором устройство слушает канал и, в случае коллизии, прекращает передачу и повторяет её позже. Используется в проводных сетях (Ethernet).

CSMA/CA (Carrier Sense Multiple Access with Collision Avoidance) — метод, при котором устройство пытается избежать коллизии заранее, посылая запрос на передачу. Используется в беспроводных сетях (Wi-Fi).

Отличие: CSMA/CD обнаруживает коллизии и реагирует после их возникновения, CSMA/CA предотвращает коллизии до начала передачи.

3. Дайте определение понятию «буфер обмена». Охарактеризуйте его назначение. Перечислите стандартные горячие клавиши для работы с буфером обмена для ОС Windows

Буфер обмена — это область памяти, в которую временно помещаются данные при копировании или вырезании, чтобы затем вставить их в другое место.

Назначение: обмен данными между приложениями и частями документа.

Горячие клавиши Windows:

  • Ctrl + C — копировать
  • Ctrl + X — вырезать
  • Ctrl + V — вставить
