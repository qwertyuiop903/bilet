Билет 8

1. Дайте определение понятию «беспроводная вычислительная сеть». Опишите принцип работы, проблемы и ограничения беспроводных сетей. Перечислите и опишите типы беспроводных сетей

Беспроводная вычислительная сеть — это сеть, в которой передача данных осуществляется без физического соединения, с использованием радиоволн, инфракрасных или микроволновых сигналов.

Принцип работы: устройства обмениваются данными через точку доступа или напрямую друг с другом с использованием протоколов беспроводной связи (например, Wi-Fi, Bluetooth).

Проблемы и ограничения:

  • Зависимость от радиопомех
  • Ограниченный радиус действия
  • Проблемы с безопасностью
  • Низкая стабильность по сравнению с проводными сетями

Типы беспроводных сетей:

  • WPAN (личная) — короткий радиус (например, Bluetooth)
  • WLAN (локальная) — Wi-Fi в пределах помещения
  • WMAN (городская) — например, WiMAX
  • WWAN (глобальная) — сотовые сети (3G, 4G, 5G)

2. Дайте определение понятию «тип данных». Перечислите и опишите типы данных в языке SQL

Тип данных — это характеристика, определяющая, какие значения может принимать переменная или поле и какие операции с ним допустимы.

Типы данных в SQL:

  • Числовые: INT, SMALLINT, BIGINT, DECIMAL, FLOAT
  • Строковые: CHAR, VARCHAR, TEXT
  • Дата и время: DATE, TIME, DATETIME, TIMESTAMP
  • Логические: BOOLEAN (TRUE / FALSE)

3. Дайте определение понятию «манипулятор типа «мышь». Перечислите и охарактеризуйте виды данных манипуляторов по конструктивным особенностям

Манипулятор типа «мышь» — это устройство ввода, предназначенное для управления курсором на экране компьютера.

Виды мышей по конструктивным особенностям:

  • Механическая — использует шарик, движущийся по поверхности
  • Оптическая — использует светодиод и датчики
  • Лазерная — более точная, использует лазер
  • Беспроводная — соединяется по радиоканалу (RF) или Bluetooth
  • Сенсорная — не имеет физических кнопок, реагирует на касания
