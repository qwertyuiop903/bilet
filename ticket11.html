Билет 11

1. Дайте характеристику SQL. В чем его отличие от языков структурного программирования

SQL (Structured Query Language) — язык структурированных запросов, используемый для работы с базами данных. Он позволяет выполнять операции выборки, обновления, добавления и удаления данных, а также управлять структурами БД.

Отличие от языков структурного программирования:

  • SQL — декларативный язык: указывается, что нужно получить, а не как это сделать
  • Структурные языки (например, C, Pascal) — императивные: описывают последовательность действий
  • SQL применяется только для взаимодействия с БД, а языки программирования — для построения полноценных программ

2. Охарактеризуйте кабель типа «витая пара»: область использования, устройство и основные свойства. Сравните его достоинства и недостатки с другими кабелями

Витая пара — тип кабеля, состоящий из пар изолированных медных проводов, скрученных между собой.

Устройство: пары проводов с определённой частотой скручивания, экранированные или неэкранированные.

Области применения: локальные сети Ethernet, телефония, системы видеонаблюдения.

Основные свойства:

  • Скорость передачи: до 10 Гбит/с (в зависимости от категории)
  • Дальность: до 100 м без усилителей
  • Стоимость: низкая

Преимущества:

  • Низкая стоимость
  • Гибкость и лёгкость прокладки

Недостатки по сравнению с другими кабелями:

  • Низкая устойчивость к помехам (в сравнении с коаксиалом или оптикой)
  • Ограниченная длина линии

3. Дайте определение понятиям «хеширование», «хеш-функция», «хеш». Приведите примеры хеш-функций и охарактеризуйте их

Хеширование — процесс преобразования входных данных произвольной длины в фиксированную последовательность (хеш).

Хеш-функция — алгоритм, выполняющий хеширование.

Хеш — результат работы хеш-функции.

Примеры хеш-функций:

  • MD5 — устаревшая, 128 бит, используется для проверки целостности
  • SHA-1 — 160 бит, также считается небезопасной
  • SHA-256 — современная, используется в криптографии

Характеристики хорошей хеш-функции:

  • Односторонность (невозможно восстановить оригинал)
  • Равномерное распределение
  • Устойчивость к коллизиям
