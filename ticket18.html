Билет 18

1. Дайте определение понятию «интерфейс ОС». Перечислите и охарактеризуйте основные элементы графического интерфейса ОС Windows.

Интерфейс операционной системы (ОС) — совокупность средств и способов взаимодействия пользователя с компьютерной системой.

Основные элементы графического интерфейса Windows:

  • Рабочий стол — основная область экрана с иконками и фоновым изображением.
  • Панель задач — строка для быстрого запуска программ, отображения открытых окон и системных уведомлений.
  • Меню «Пуск» — центральное меню для доступа к программам, настройкам и выключению системы.
  • Окна — область отображения приложений и документов с заголовками и элементами управления.
  • Иконки — графические изображения для быстрого доступа к файлам, папкам и программам.
  • Диалоговые окна — окна для взаимодействия пользователя с программами (например, запросы на ввод).

2. Укажите назначение команды ALTER TABLE. Перечислите способы применения ALTER TABLE для изменения таблицы.

ALTER TABLE — команда SQL для изменения структуры существующей таблицы в базе данных.

Способы применения ALTER TABLE:

  • Добавление столбца: ALTER TABLE имя_таблицы ADD имя_столбца тип_данных;
  • Удаление столбца: ALTER TABLE имя_таблицы DROP COLUMN имя_столбца;
  • Изменение типа столбца: ALTER TABLE имя_таблицы ALTER COLUMN имя_столбца TYPE новый_тип;
  • Переименование столбца: ALTER TABLE имя_таблицы RENAME COLUMN старое_имя TO новое_имя;
  • Добавление ограничений (constraints): например, уникальность или внешний ключ.

3. Дайте определение понятию «асимметричная криптосистема». Охарактеризуйте алгоритм шифрования RSA и схему Эль-Гамаля.

Асимметричная криптосистема — система шифрования, использующая пару ключей: публичный (открытый) для шифрования и приватный (секретный) для дешифрования.

Алгоритм RSA: основан на трудности факторизации больших чисел. Пользователь генерирует пару ключей, публичный используется для шифрования, приватный — для расшифровки. Применяется для безопасной передачи данных и цифровых подписей.

Схема Эль-Гамаля: основана на дискретном логарифме. Использует случайные числа и вычисления по модулю для обеспечения безопасности. Позволяет выполнять шифрование и цифровую подпись с высокой степенью защиты.
