1. Опишите принцип работы, преимущества и недостатки статической маршрутизации. Приведите примеры протоколов статической маршрутизации и охарактеризуйте их

Статическая маршрутизация — это ручная настройка маршрутов администратором без участия протоколов динамической маршрутизации.

Преимущества:

  • Простота настройки в небольших сетях
  • Высокая производительность, так как не требуется обмен маршрутной информацией

Недостатки:

  • Сложность масштабирования
  • Неавтоматическое восстановление маршрутов при сбоях

Примеры:

  • Static Routing — вручную добавляемые маршруты с помощью команды `ip route`
  • Policy-Based Routing — маршрутизация на основе политики, а не только IP

2. Дайте определение понятию «администрирование БД». Перечислите и опишите задачи администратора БД

Администрирование БД — это процесс управления и поддержки баз данных с целью обеспечения их надёжности, производительности и безопасности.

Основные задачи администратора:

  • Установка и настройка СУБД
  • Управление пользователями и правами доступа
  • Резервное копирование и восстановление данных
  • Оптимизация производительности
  • Мониторинг состояния базы
  • Обеспечение безопасности данных

3. Укажите назначение форм на веб-страницах. Перечислите и охарактеризуйте управляющие элементы для создания форм с помощью HTML

Формы используются для ввода и отправки данных пользователем на сервер.

Основные управляющие элементы:

  • <input type="text"> — поле ввода текста
  • <input type="password"> — поле для ввода пароля
  • <input type="checkbox"> — флажок выбора
  • <input type="radio"> — переключатель
  • <select> и <option> — выпадающий список
  • <textarea> — многострочное текстовое поле
  • <button> и <input type="submit"> — кнопка отправки формы
