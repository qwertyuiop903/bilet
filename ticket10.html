Билет 10

1. Укажите основные команды манипулирования данными в SQL. Укажите и охарактеризуйте основные элементы синтаксиса данных команд

Команды манипулирования данными (DML) позволяют работать с содержимым таблиц:

  • SELECT — выборка данных
  • INSERT — добавление новых записей
  • UPDATE — изменение существующих записей
  • DELETE — удаление записей

Основные элементы синтаксиса:

  • WHERE — фильтрация по условиям
  • GROUP BY — группировка
  • HAVING — фильтрация после группировки
  • ORDER BY — сортировка результатов
  • JOIN — объединение таблиц

2. Дайте определение понятию «сетевая модель OSI». Перечислите и укажите функции уровней сетевой модели OSI

Модель OSI (Open Systems Interconnection) — эталонная сетевая модель, состоящая из 7 уровней, каждый из которых выполняет определённые функции при передаче данных.

  • 1. Физический уровень — передача битов по физической среде
  • 2. Канальный уровень — определение границ кадров, контроль ошибок
  • 3. Сетевой уровень — маршрутизация и адресация (IP)
  • 4. Транспортный уровень — надежность доставки (TCP, UDP)
  • 5. Сеансовый уровень — управление сеансами связи
  • 6. Представительный уровень — преобразование форматов данных
  • 7. Прикладной уровень — взаимодействие с пользовательскими приложениями

3. Дайте определение понятиям «брандмауэр», «антивирусная программа». Приведите классификацию антивирусных программ. Перечислите известные антивирусные программы и их функции

Брандмауэр (Firewall) — средство защиты, контролирующее сетевой трафик на основе заданных правил.

Антивирусная программа — ПО для обнаружения, удаления и предотвращения вредоносного ПО.

Классификация антивирусов:

  • Сигнатурные
  • Проактивные (поведенческие)
  • Облачные
  • Комплексные (интернет-секьюрити)

Известные антивирусы:

  • Kaspersky — защита в реальном времени, контроль приложений
  • Avast — облачный анализ, веб-защита
  • Norton — защита от фишинга, резервное копирование
  • Dr.Web — лечение заражённых систем
  • Bitdefender — антишпион, родительский контроль
