Билет 9

1. Дайте определение понятию «интерфейс ОС». Перечислите и охарактеризуйте основные элементы графического интерфейса ОС Windows

Интерфейс операционной системы — это средство взаимодействия пользователя с ОС, предоставляющее доступ к функциям системы и приложениям.

Элементы графического интерфейса Windows:

  • Рабочий стол — основное пространство с ярлыками и фоном
  • Панель задач — содержит меню «Пуск», открытые окна, часы и уведомления
  • Окна — рамки программ с заголовком, кнопками управления
  • Меню «Пуск» — центральная точка запуска программ и поиска
  • Проводник — инструмент для работы с файлами и папками

2. Укажите назначение команды ALTER TABLE. Перечислите способы применения ALTER TABLE для изменения таблицы

ALTER TABLE — команда SQL, используемая для изменения структуры существующей таблицы.

Применения ALTER TABLE:

  • Добавление столбца: ADD COLUMN
  • Удаление столбца: DROP COLUMN
  • Изменение типа столбца: MODIFY / ALTER COLUMN
  • Переименование столбца: RENAME COLUMN
  • Добавление/удаление ограничений: ADD/DROP CONSTRAINT

3. Дайте определение понятию «асимметричная криптосистема». Охарактеризуйте алгоритм шифрования RSA и схему Эль-Гамаля

Асимметричная криптосистема — это метод шифрования, в котором используются два ключа: публичный для шифрования и приватный для расшифровки.

RSA:

  • Основывается на сложности факторизации больших чисел
  • Ключи формируются с использованием двух больших простых чисел
  • Широко применяется для шифрования и цифровых подписей

Схема Эль-Гамаля:

  • Основана на трудности вычисления дискретного логарифма
  • Генерируется пара ключей: открытый и закрытый
  • Шифрование сопровождается рандомизацией, что делает один и тот же текст разным при повторном шифровании
