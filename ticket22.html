1. Дайте определение понятию NAT. Опишите принцип работы NAT. Перечислите и охарактеризуйте типы NAT.

NAT (Network Address Translation) — технология преобразования сетевых адресов, позволяющая множеству устройств в локальной сети использовать один публичный IP-адрес для выхода в интернет.

Принцип работы: NAT-устройство (обычно маршрутизатор) заменяет приватные IP-адреса внутренних устройств на публичный IP-адрес при передаче данных во внешнюю сеть и обратно, отслеживая соединения с помощью таблицы трансляции.

Типы NAT:

  • Статический NAT — постоянное сопоставление одного внутреннего адреса с одним внешним (1:1).
  • Динамический NAT — пул публичных адресов распределяется между внутренними устройствами по мере необходимости.
  • PAT (Port Address Translation) или NAT Overload — несколько внутренних адресов преобразуются в один публичный с использованием разных портов.
  • NAT с перекрытием — когда внутренние и внешние сети используют одинаковые диапазоны IP-адресов.

2. Дайте определение понятию «DLL (Dynamic Link Library)». Опишите структуру DLL. Укажите основные утилиты для работы с DLL и их функционал.

DLL (Dynamic Link Library) — динамически подключаемая библиотека в Windows, содержащая код и данные, которые могут использоваться несколькими программами одновременно.

Структура DLL:

  • Заголовок (PE-формат) с метаданными
  • Таблица экспорта (функции, доступные другим программам)
  • Таблица импорта (внешние зависимости)
  • Секции кода (.text), данных (.data) и ресурсов (.rsrc)
  • Таблица перемещений (relocations)

Утилиты для работы с DLL:

  • Dependency Walker — анализ зависимостей DLL, просмотр экспортируемых/импортируемых функций
  • DLL Export Viewer — просмотр экспортируемых функций и их ординалов
  • PE Explorer — анализ структуры PE-файлов, включая DLL
  • dumpbin (из Visual Studio) — анализ содержимого DLL из командной строки
  • regsvr32 — регистрация и отмена регистрации DLL в системе

3. Дайте определение понятиям «хакерская атака», «эксплойт». Перечислите и охарактеризуйте технологии обнаружения атак.

Хакерская атака — умышленное действие, направленное на нарушение конфиденциальности, целостности или доступности информационной системы.

Эксплойт — фрагмент программного кода или последовательность команд, использующие уязвимости в программном обеспечении для проведения атаки.

Технологии обнаружения атак:

  • IDS (Intrusion Detection System) — системы обнаружения вторжений, анализирующие сетевой трафик или события на хостах
  • IPS (Intrusion Prevention System) — системы предотвращения вторжений, активно блокирующие атаки
  • SIEM (Security Information and Event Management) — комплексные системы сбора и анализа событий безопасности
  • Анализ сигнатур — поиск известных шаблонов атак
  • Анализ аномалий — выявление отклонений от нормального поведения
  • Песочницы (Sandboxing) — изолированная среда для анализа подозрительных объектов
  • Honeypot — ловушки для привлечения и изучения атакующих
