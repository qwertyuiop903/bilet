Билет 6

1. Дайте определение следующим понятиям «физическая модель диска», «логическая модель диска». Перечислите папки в корневом каталоге Linux и укажите, какие данные они хранят

Физическая модель диска — описание устройства хранения данных на физическом уровне (например, секторы, дорожки, цилиндры).

Логическая модель диска — организация данных с точки зрения операционной системы (файловая система, папки, файлы).

Основные папки в корне Linux:

  • /bin — основные исполняемые файлы
  • /boot — загрузочные файлы
  • /etc — конфигурационные файлы
  • /home — домашние каталоги пользователей
  • /lib — библиотеки, используемые системными программами
  • /media — точки монтирования съемных носителей
  • /opt — сторонние приложения
  • /root — домашняя директория суперпользователя
  • /sbin — системные утилиты
  • /tmp — временные файлы
  • /usr — пользовательские приложения
  • /var — переменные файлы, логи

2. Дайте определение понятию «HTML». Перечислите и опишите содержимое основных элементов структуры HTML-документа

HTML (HyperText Markup Language) — язык разметки, используемый для создания структуры веб-страниц.

Основные элементы HTML-документа:

  • <!DOCTYPE html> — определяет тип документа
  • <html> — корневой элемент документа
  • <head> — содержит метаданные (кодировка, заголовок, стили, скрипты)
  • <title> — заголовок страницы в браузере
  • <body> — содержимое страницы, отображаемое пользователю

3. Дайте определение понятию «цифровая подпись». Охарактеризуйте причины ее использования

Цифровая подпись — это криптографический механизм, подтверждающий подлинность и целостность электронного документа или сообщения.

Основные причины использования:

  • Аутентификация — проверка отправителя
  • Целостность — защита от изменений данных
  • Неподделываемость — гарантия, что сообщение подписано конкретным лицом
  • Юридическая значимость — использование в электронных сделках и документах
