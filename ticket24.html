1. Дайте определение понятиям «каскадная таблица стилей (CSS)», «селектор». Опишите назначение CSS. Перечислите виды селекторов.

CSS (Cascading Style Sheets) — язык стилей, определяющий представление HTML-документов, включая дизайн, макет и вариации отображения для различных устройств.

Селектор — шаблон для выбора элементов HTML, к которым будут применены стилевые правила.

Назначение CSS:

  • Отделение содержания (HTML) от представления (стилей)
  • Управление внешним видом веб-страниц
  • Создание адаптивных дизайнов для разных устройств
  • Упрощение поддержки и изменения дизайна сайта
  • Повышение скорости загрузки страниц

Виды селекторов:

  • Универсальный (*) — выбирает все элементы
  • По тегу (div, p) — выбирает элементы по имени тега
  • По классу (.classname) — выбирает элементы с указанным классом
  • По идентификатору (#idname) — выбирает элемент с указанным id
  • По атрибуту ([attr="value"]) — выбирает элементы по атрибуту
  • Псевдоклассы (:hover, :focus) — выбирают элементы в особом состоянии
  • Псевдоэлементы (::before, ::after) — выбирают часть элемента
  • Комбинаторы (пробел, >, +, ~) — выбирают элементы по их положению

2. Что такое «маркер доступа» и «проверка прав доступа»? Укажите ключевые компоненты маркера доступа и опишите основные его типы.

Маркер доступа (Access Token) — объект или структура данных, содержащая информацию о правах доступа субъекта (пользователя, процесса) к объектам системы.

Проверка прав доступа — процесс верификации, имеет ли субъект необходимые права для выполнения запрошенной операции над объектом.

Ключевые компоненты маркера доступа:

  • Идентификатор субъекта (SID)
  • Идентификатор группы/роли
  • Привилегии и права доступа
  • Временные параметры (срок действия)
  • Цифровая подпись (для проверки подлинности)

Основные типы маркеров доступа:

  • Дискреционный (DAC) — владелец ресурса определяет права доступа
  • Мандатный (MAC) — права назначаются системой на основе политик безопасности
  • Ролевой (RBAC) — доступ предоставляется на основе ролей пользователя
  • На основе атрибутов (ABAC) — доступ определяется атрибутами субъекта, ресурса и окружения
  • OAuth-токены — используются в веб-приложениях для делегированного доступа
  • JWT (JSON Web Tokens) — компактные токены в формате JSON с цифровой подписью

3. Дайте определение понятиям «криптография», «криптоанализ», «шифр». Опишите основные функции криптосистем.

Криптография — наука о методах обеспечения конфиденциальности, целостности, аутентичности и невозможности отказа от авторства информации.

Криптоанализ — наука о методах вскрытия шифров и получения исходного сообщения без знания ключа.

Шифр — алгоритм преобразования информации с целью ее сокрытия от посторонних, использующий ключ для шифрования и дешифрования.

Основные функции криптосистем:

  • Конфиденциальность — защита от несанкционированного доступа
  • Целостность — обнаружение любых изменений данных
  • Аутентификация — подтверждение подлинности участников обмена
  • Неотрекаемость — невозможность отказа от совершенных действий
  • Управление ключами — генерация, распределение, хранение и уничтожение ключей
  • Стойкость — сопротивление криптоанализу и атакам
  • Эффективность — разумное соотношение безопасности и производительности
